# Task: Replace tauri-typegen with typeshare — annotate structs, delete commands.ts, update scripts

## Context

tauri-typegen generates `src/generated/types.ts` and `src/generated/commands.ts` from Tauri command signatures. Has history of wrong output (bad serde rename, mismatched types). Task removes it entirely. typeshare replaces for type gen; `invoke()` calls move into hooks that use them; `commands.ts` deleted.

Runs after 001–003 — needs final Rust struct shapes stable before annotating.
