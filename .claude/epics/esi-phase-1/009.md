# Task: Frontend event listener bootstrap

## Description

Create `src/lib/esiEvents.ts` that registers Tauri event listeners for all `character:*:*` events and `notifications:new`. Each listener deserializes the payload (typed against `src/generated/events.ts`) and calls `queryClient.setQueryData(queryKey, payload.data)` to update the React Query cache. Call `bootstrapEsiEvents(queryClient)` on app mount. This is the Phase 1 frontend adapter — it will be replaced by Zustand store setters in Phase 2.

## Acceptance Criteria

- [ ] `src/lib/esiEvents.ts` created; exports `bootstrapEsiEvents(queryClient: QueryClient): Promise<void>`
- [ ] Registers listeners for: `character:*:queue`, `character:*:skills`, `character:*:location`, `character:*:public`, `character:*:attributes`, `character:*:clones`, `character:*:remaps`, `notifications:new`
- [ ] Event name pattern parsed to extract `character_id` from the event name string
- [ ] Payload types imported from `src/generated/events.ts` (generated by typeshare in Task 002)
- [ ] Each listener calls `queryClient.setQueryData` with the correct existing query key from `queryKeys.ts`
- [ ] `bootstrapEsiEvents` called in `src/routes/__root.tsx` (or `src/main.tsx`) on app mount after QueryClient is available
- [ ] Also calls `get_esi_snapshot` for each known character on mount and seeds cache
- [ ] No component-level event subscriptions — all listeners registered in this single bootstrap

## Technical Details

```typescript
import { listen } from '@tauri-apps/api/event';
import type { QueuePayload } from '../generated/events';

export async function bootstrapEsiEvents(queryClient: QueryClient) {
  await listen<QueuePayload>('character:*:queue', (event) => {
    const { character_id, queue } = event.payload;
    queryClient.setQueryData(queryKeys.skillQueue(character_id), queue);
  });
  // ... repeat for each resource
}
```

Note: Tauri v2 glob event listeners — verify `character:*:queue` wildcard syntax works in Tauri v2. If not, register listeners with the full event name after fetching known character IDs.
