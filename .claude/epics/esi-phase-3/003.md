# Task: Remove leftover RQ live-polling config

## Description

Strip any `refetchInterval`, `staleTime`, or `gcTime` configuration that was tuned for live ESI polling from hooks that have been migrated to Zustand. These settings are dead weight now that Zustand owns the live data layer.

- [ ] No `refetchInterval` in `src/hooks/tauri/` for ESI live-data hooks

```bash
grep -rn "refetchInterval" src/hooks/tauri/
grep -rn "staleTime" src/hooks/tauri/
```

Hooks that are still RQ-based (mutations, SDE, settings) may retain their own `staleTime`/`gcTime` — only remove config from formerly live-polling hooks.
